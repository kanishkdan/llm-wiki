# OpenClaw

OpenClaw is an open-source framework for building personal AI agents. Kanishk deployed his instance, named [[concepts/digital-operator|Rudra]], on a MacBook Air to ensure local access to the filesystem and terminal.

## Architecture
- **Local execution:** Runs on macOS for direct data access.
- **Interface:** Controlled via a Telegram bot for mobile-first interaction.
- **Workflow:** Integrated with Git and GitHub for automated blog publishing.

## Implementation: Rudra
Rudra serves as Kanishk's "Digital Operator," handling tasks ranging from coding to content generation.
